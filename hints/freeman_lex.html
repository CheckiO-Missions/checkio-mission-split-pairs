That's almost all. Just add function statement, return 'result' and import Iterable from 'typing' module to proper typehints work: 

from typing import Iterable

def split_pairs(text: str) -> Iterable[str]:

    if len(text) % 2:
        text += '_'
        
    result: list = []
    while text:
        result.append(text[:2])
        text = text[2:]
    
    return result
There are a lot of other solutions you be able to see after your solution passes all tests successfully. For example: 

split_pairs = lambda a: [i + k for i, k in zip(a[::2], a[1::2] + '_')]
Zipping (taking values on each index) lists with even and odd characters together inside list comprehension. Zipping ends with shorter sequence. Or: 

from typing import Iterable

def split_pairs(text: str) -> Iterable[str]:

    text += '_' if len(text) % 2 else ''

    return [text[i:i + 2] for i in range(0, len(text), 2)]
