That's almost all. Just add function statement, return result and import Iterable from typing module to proper typehints work: 

from collections.abc import Iterable


def split_pairs(text: str) -> Iterable[str]:

    if len(text) % 2:
        text += '_'
        
    result: list = []
    while text:
        result.append(text[:2])
        text = text[2:]
    
    return result

Zipping (taking values on each index) lists with even and odd characters together inside list comprehension. Zipping ends with shorter sequence. 

split_pairs = lambda a: [i + k for i, k in zip(a[::2], a[1::2] + '_')]

Using one range and taking slices. Second index is not included, so i + 2. 

from collections.abc import Iterable


def split_pairs(text: str) -> Iterable[str]:

    text += '_' if len(text) % 2 else ''

    return [text[i:i + 2] for i in range(0, len(text), 2)]
