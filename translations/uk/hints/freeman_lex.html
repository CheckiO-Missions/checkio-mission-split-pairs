Я не маю уявлення, як розв'язувати цю місію.
Ідея у використанні 'зрізів'. Береш два перші символи з тексту, робиш текст коротшим і повторюєш.
З чого мені слід почати?
Твій перший крок - перевірити чи текст має парну кількість символів. Зроби це з використанням умови if, функції len() і оператора %. Якщо довжина тексту непарна, додай символ підкреслення: 

if len(text) % 2:
    text += '_'
Ці команди можуть бути записані іншими способами, поки можливо менш зрозумілими: 

text += '_' * (len(text) % 2) # множення
# або
text += ('', '_')[len(text) % 2] # індексування
# і т.д.
Добре, це лише початок. Що мені робити далі?
Тепер тобі слід проголосити змінну 'result', значення якої ти повернеш в кінці. Тип змінної має бути одним із ітеративних об'єктів, тож це може бути список. Проголоси змінну з використанням підказок типу змінної або без: 

result: list[str] = []    
# або        
result: list = []
# або просто
result = []
Це зроблене, але я не розумію, як виконати основну роботу.
Використай метод list.append() і 'зрізи', щоб додати перші два символи тексту до списку 'result' і оновити текст, щоб він став коротшим на цих два символи: 

if len(text) % 2:
    text += '_'
    
result: list = []
# команда, яку ми додамо пізніше
    result.append(text[:2])
    text = text[2:]
Я розумію, але ж це лише один крок. Як робити ці операції з усім текстом?
Оскільки ти робиш текст коротшим з кожним кроком, то повторюй це, поки текст не стане порожнім з циклом while: 

if len(text) % 2:
    text += '_'
    
result: list = []
while text:
    result.append(text[:2])
    text = text[2:]
Добре, як має виглядати остаточне рішення?
Це і так майже все. Просто додай декларування самої функції, поверни значення змінної 'result' і імпортуй Iterable з модуля typing для коректної роботи підказок типу змінної: 

from typing import Iterable


def split_pairs(text: str) -> Iterable[str]:

    if len(text) % 2:
        text += '_'
        
    result: list = []
    while text:
        result.append(text[:2])
        text = text[2:]
    
    return result

Звісно, тримай! 

split_pairs = lambda a: [i + k for i, k in zip(a[::2], a[1::2] + '_')]
Зіпування (беруться значення кожного індексу з усіх елементів) списків з парними і непарними символами разом всередині генератора списків. Зіпування закінчується із завершенням коротшого списку. 

from typing import Iterable


def split_pairs(text: str) -> Iterable[str]:

    text += '_' if len(text) % 2 else ''

    return [text[i:i + 2] for i in range(0, len(text), 2)]
Використання діапазону і зрізу. Другий індекс в діапазоні не включається, тому 'i + 2'.
