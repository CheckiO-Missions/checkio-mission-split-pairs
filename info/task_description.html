Split the string into pairs of two characters. If the string contains an odd number of characters, then the missing second character of the final pair should be replaced with an underscore ('_').

Input: A string.

Output: {% if interpreter.slug == "js-node" %}An array of strings{% else %}An iterable of strings{% endif %}.

Example:

{% if interpreter.slug == "js-node" %} 
splitPairs('abcd') == ['ab', 'cd']
splitPairs('abc') == ['ab', 'c_']
splitPairs('abcdf') == ['ab', 'cd', 'f_']
splitPairs('a') == ['a_']
splitPairs('') == []
{% else %} 
split_pairs('abcd') == ['ab', 'cd']
split_pairs('abc') == ['ab', 'c_']
{% endif %}

Precondition: 0<=len(str)<=100
