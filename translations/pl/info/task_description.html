Podziel string na pary znaków. Jeśli string zawiera nieparzystą liczbę znaków, brakujący znak w ostatniej parze należy zastąpić podkreślnikiem ('_').

Wejście: String.

Wyjście: {% if interpreter.slug == "js-node" %}Tablica stringów (array of strings){% else %}Iterable zawierający stringi (iterable of strings){% endif %}.

Przykład:

{% if interpreter.slug == "js-node" %} 
splitPairs('abcd') == ['ab', 'cd']
splitPairs('abc') == ['ab', 'c_']
splitPairs('abcdf') == ['ab', 'cd', 'f_']
splitPairs('a') == ['a_']
splitPairs('') == []
{% else %} 
split_pairs('abcd') == ['ab', 'cd']
split_pairs('abc') == ['ab', 'c_']
{% endif %}

Warunek wstępny: 0<=len(str)<=100
